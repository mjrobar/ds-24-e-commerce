# DS-24 E-commerce

Modern, responsive e-commerce platform built with Vite.

## Features
- Multi-page application structure.
- Dynamic product filtering and search.
- Integrated cart and checkout flow.
- Glassmorphism design aesthetics.
- Gemini AI integration readiness.

## Getting Started

**Prerequisites:** Node.js

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Configuration**: Create a `.env` file and add your `GEMINI_API_KEY`.
3.  **Run Locally**:
    ```bash
    npm run dev
    ```

## Deployment

This project is ready for deployment on Vercel. Connect your GitHub repository to Vercel and ensure you add the `GEMINI_API_KEY` to your environment variables.
